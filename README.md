# 😂 Каталог анекдотов

Это мой первый веб-сайт с анекдотами.

## 📌 Что умеет сайт:
- Выбор категорий анекдотов
- Случайный анекдот по кнопке
- Категория "Общее" (все анекдоты сразу)
- Красивый фон и дизайн
- Простая и удобная работа

## 💻 Использованные технологии:
- HTML
- CSS
- JavaScript
